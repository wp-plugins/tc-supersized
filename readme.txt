Fullscreen background image changer with dynamic transition based on supersized.js

== Description ==

TC Supersized give you feature to change your website background dynamically with slideshow transition based on supersized.js.With easy to use and less configuration .Provide up to 10 images to add or less as your website background.



Current Feature : "Transition" , "10 images for slideshow"

== Installation ==

1. Upload 'TC Supersized' to the '/wp-content/plugins/' directory,
2. Activate the plugin through the 'Plugins' menu in WordPress.
